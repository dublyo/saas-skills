## Product-Growth Rule

Marketing is not only traffic. Many SaaS channels need product surfaces:

- Free tools.
- Public reports.
- Share links.
- Badges.
- Templates.
- Review prompts.
- Invite flows.
- Lifecycle emails.
- Partner/reseller accounts.
- Affiliate attribution.
- White-label settings.
- Integration marketplace pages.
- Analytics and attribution events.

Build only the loops that match the product stage and ICP.

## SEO

Content SEO:

- Educational articles.
- Problem/solution pages.
- Use-case guides.
- Industry pages.
- Tutorials.

Commercial SEO:

- Alternative pages.
- Comparison pages.
- "Best X for Y" pages.
- Pricing and feature pages.
- Competitor migration pages.

Programmatic SEO:

- Pages by platform, industry, region, template, integration, or use case.
- Use only when the product has enough structured data to avoid thin pages.

Technical SEO:

- Fast pages, semantic HTML, metadata, sitemap, robots, canonical URLs, structured data, internal links, Core Web Vitals.

Development hooks:

- CMS or content model.
- Programmatic page generator.
- Metadata system.
- Sitemap generation.
- Structured data.
- Search-console-friendly URLs.

## AI Search Optimization, GEO, And AEO

Optimize for AI answer engines and search summaries:

- Direct answer blocks.
- FAQ sections.
- Comparison tables.
- Clear definitions.
- Stats, case studies, and sourceable claims.
- Public docs and API docs.
- Review profiles and independent mentions.
- Schema markup where useful.

Development hooks:

- Public docs.
- Clear landing/comparison pages.
- Public changelog.
- Public examples/reports.
- Customer story pages.

## Product-Led Growth

PLG works when the user can experience value without a salesperson.

Examples:

- Free trial.
- Freemium plan.
- Free scan/check/report.
- Sample output.
- Interactive demo.
- Shareable report.
- Public status page.
- Powered-by badge.
- Export with branding.

Typical flow:

```text
Visitor enters URL or data
-> receives useful partial output
-> sees next unlocked value
-> signs up or upgrades
```

Development hooks:

- Anonymous or low-friction entry flow.
- Result generation before signup when safe.
- Email capture.
- Upgrade prompts tied to real limits.
- Activation events.
- Shareable artifact.

## Creator, Influencer, And Community Growth

Good partners:

- Micro-influencers.
- Specialist YouTubers.
- Newsletter writers.
- Trainers.
- SEO experts.
- WordPress/Shopify developers.
- Agencies.
- No-code builders.

Collaboration formats:

- Review, tutorial, sponsored video, case study, affiliate deal, webinar, newsletter feature, co-created resource.

Community-led rule:

- Build around the problem, not only the product.
- Avoid spam. Lead with useful audits, examples, templates, and answers.

Development hooks:

- Partner demo account.
- Co-branded report or template.
- Trackable links.
- Referral/affiliate attribution.

## Cold Outreach, Sales-Led Growth, And ABM

Cold outreach is useful early when the ICP is specific.

Channels:

- Email.
- LinkedIn.
- Direct messages.
- Phone.
- WhatsApp where appropriate and legal.

Correct approach:

- Send concrete value first.
- Reference a real problem.
- Offer a useful report or fix path.
- Avoid generic pitch copy.

Sales-led growth fits:

- Enterprise deals.
- Agencies.
- Government.
- Large teams.
- Annual contracts.
- Custom procurement.

ABM:

- Target named accounts.
- Create custom report, landing page, demo, email sequence, and follow-up.

Development hooks:

- CRM fields or export.
- Lead source tracking.
- Demo workspace.
- Custom report generator.
- Sales notes or pipeline integration.
- Enterprise plan placeholders only when needed.

## Affiliate, Referral, Partnerships, Reseller, And White-Label

Affiliate program:

- Partners: agencies, consultants, educators, creators, newsletters, SaaS directories.
- Commissions: recurring, one-time, tiered, or hybrid.

Referral program:

- Reward users for inviting qualified customers.
- Rewards can be credits, discounts, extra seats, extra usage, free month, or cash.

Partnerships:

- Technology partners.
- Service partners.
- Co-marketing.
- Agency partnerships.

Reseller and white-label:

- Agency account.
- Client management.
- Branded reports.
- Custom domain.
- Removed product branding.
- Client invitations.
- Reseller billing or seat management.

Development hooks:

- Attribution codes.
- Referral links.
- Partner dashboard.
- Commission records.
- White-label settings.
- Client/sub-account model.
- Custom branding.
- Usage and margin controls for resold plans.

## Review Platforms, Launch Platforms, And Directories

Review platforms:

- G2, Capterra, GetApp, TrustRadius, Trustpilot, AlternativeTo, SaaSworthy, and niche directories.

Needs:

- Profile, screenshots, feature list, pricing, reviews, replies, comparison pages.

Launch and deal platforms:

- Product Hunt, BetaList, Hacker News, Indie Hackers, Uneed, AppSumo, and similar.

AppSumo/deal warning:

- Define usage limits, storage limits, sites/projects, support level, add-ons, and cost ceilings. Lifetime deals can lose money if usage-heavy features are unlimited.

Development hooks:

- Review prompt timing.
- Review link tracking.
- Public roadmap/changelog.
- Deal-plan limits.
- Lifetime-plan entitlement mapping.

## Email Marketing And Lifecycle

Lead nurturing:

- For users who download a template or use a free tool.

Onboarding email sequence:

- Day 1: start first action.
- Day 2: add notification or integration.
- Day 3: discover reports.
- Day 5: invite teammate.
- Day 7: upgrade prompt tied to value.

Lifecycle triggers:

- Did not activate.
- Hit plan limit.
- Completed first report.
- Has not returned.
- Trial ending.
- Payment failed.

Retention:

- Weekly report, monthly summary, new feature, usage insights, recommendations.

Win-back:

- Cancellation reason, smaller plan, pause instead of cancel, new feature, targeted discount.

Development hooks:

- Event tracking.
- User preferences.
- Unsubscribe.
- Email templates.
- Segments.
- Deliverability setup.

## Product Virality And Sharing Loops

Build product use so it can expose the product to more users.

Examples:

- Public reports.
- Shareable dashboards.
- Status pages.
- Invite team member.
- Client report links.
- Powered-by badge.
- Embed widget.
- Website health badge.
- Shared PDF.
- Collaborative comments.

Loop:

```text
User creates useful output
-> shares it with client/team/public
-> recipient discovers product
-> recipient creates account or asks for access
```

Development hooks:

- Share permissions.
- Public/private toggle.
- Branded exports.
- Viewer analytics.
- Signup from shared artifact.
- Abuse and privacy controls.

## Practical Channel Phasing

Early, first 20 users:

- Direct outreach to exact ICP.
- Free useful audit/report.
- Founder-led content.
- Relevant communities.
- Simple free tools.
- Beta launch.

First 100 customers:

- SEO and comparison pages.
- Tutorials.
- Affiliate program.
- Agency partnerships.
- White-label plan.
- Review platforms.
- Referral program.
- Lifecycle email.

Scale:

- Paid search.
- Integration marketplaces.
- Reseller program.
- Programmatic SEO.
- Original research.
- Webinars.
- ABM for larger companies.
- Sales team.

Do not run every channel at once. Pick the smallest channel set that matches the product and can be supported by the current app.
